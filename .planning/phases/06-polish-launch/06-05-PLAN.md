Create the docs/ directory and write docs/cms-guide.md. Rules for the guide:
- Plain language only — no code, no JSON, no terminal commands
- Step-by-step numbered lists
- Written in English (for the owner)
- Covers exactly four tasks per D-14 (do not add more)
- Includes keystatic.cloud and the Keystatic Cloud login flow per D-15

The guide must contain these four sections in order:

SECTION: How to Access the CMS
1. Go to keystatic.cloud in your browser.
2. Sign in with your Keystatic Cloud account (email and password set up during site launch).
3. Once signed in, click the By Blendstrup project.
4. The admin panel has two sections on the left: Pieces (ceramic works) and Settings
   (homepage and site settings).

SECTION: Task 1 — Adding a New Ceramic Piece
Use this whenever you have a new piece to show on the website.
1. In the left sidebar, click Pieces.
2. Click the Add item (or New piece) button in the top right.
3. Fill in the fields:
   - Title: The name of the piece (e.g., "Stoneware bowl, autumn glaze").
   - Images: Click Add image and upload a photo from your computer. The first photo
     is used as the main image. Add a short description in the Alt text field —
     e.g., "Stoneware bowl with matte grey glaze". This helps search engines.
   - Description: A short text about the piece — material, size, what makes it
     special. This appears on the piece's detail page.
   - Category: Choose the most fitting category from the dropdown.
   - Sale status: Choose Available (for sale), Not listed (not for sale), or Sold.
   - Price: Enter the price as text, e.g., "1.200 kr.". Leave blank if not for sale.
   - Lead time: A note about delivery, e.g., "Ships within 1 week". Leave blank
     if not for sale.
   - Published: Toggle to Published when ready. Leave as Draft while preparing.
4. Click Save. The website rebuilds automatically within a few minutes.

SECTION: Task 2 — Marking a Piece as Sold
Use this when a piece has been purchased.
1. In the left sidebar, click Pieces.
2. Find the piece in the list and click on it.
3. Change Sale status from Available to Sold.
4. Click Save. The piece will now show a "Sold" label and the "Contact to buy" button
   will disappear.

SECTION: Task 3 — Curating the Homepage
Use this to control which pieces appear in the hero image and shop preview.
1. In the left sidebar, click Homepage (under Settings or Singletons).
2. Hero piece: Select which piece appears as the large hero image at the top.
3. Shop preview pieces: Select which pieces appear in the "For sale" preview section.
   You can select multiple.
4. Click Save. The homepage will update within a few minutes.

SECTION: Task 4 — Editing Site Settings
Use this to update your contact email or Instagram handle.
1. In the left sidebar, click Settings (under Singletons or Site settings).
2. Contact email: Update the email address that purchase and custom order inquiries
   are sent to.
3. Instagram handle: Update your Instagram username (without the @ symbol).
4. Click Save.

SECTION: Tips
- Changes take a few minutes: Keystatic commits to GitHub and Vercel rebuilds the site.
  This usually takes 1-3 minutes.
- Drafts are safe: A piece with Published turned off will never appear on the website.
- Image sizes: Upload photos at their original resolution — the website optimises them.
- Need help? Contact your developer.

Write these sections with appropriate markdown headings (## for top-level sections,
no horizontal rule dividers between sections). The guide should read naturally as a
plain-text document.

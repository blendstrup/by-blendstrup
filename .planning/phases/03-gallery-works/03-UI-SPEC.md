## Spacing Scale

Inherited from Phase 1 — no new values. All spacing uses the 4px grid.

| Token | Value | Tailwind class | Usage in this phase |
|-------|-------|----------------|---------------------|
| xs | 4px | `gap-1` / `p-1` | Badge internal padding |
| sm | 8px | `gap-2` / `p-2` | Badge text gap, filter button gap |
| md | 16px | `gap-4` / `p-4` | Filter button padding |
| lg | 24px | `gap-6` / `p-6` | Card gap (mobile/tablet grid) |
| xl | 32px | `gap-8` / `p-8` | Detail page column gap |
| 2xl | 48px | `gap-12` / `p-12` | Section padding, horizontal page margins (mobile) |
| 3xl | 64px | `gap-16` / `p-16` | Horizontal page margins (desktop), grid gap (desktop) |
| 4xl | 96px | `gap-24` / `p-24` | Major vertical section breaks |

Exceptions:
- Grid card image: no internal padding — image bleeds to card edge.
- Status badge: `px-2 py-1` (8px × 4px) — tighter than standard md to keep badge compact on image corner.
- Touch target for filter toggle buttons: minimum 44px height (WCAG 2.5.5 AA).

Source: Phase 1 UI-SPEC `## Spacing Scale`. No new values added.

## Typography

Inherited from Phase 1 — no new sizes or weights. Four roles, two weights.

| Role | Typeface | Size | Weight | Line Height | Tailwind class | Phase 3 usage |
|------|----------|------|--------|-------------|----------------|---------------|
| Body | DM Sans | 16px | 400 | 1.6 | `text-base font-normal` | Piece description, filter labels |
| Label / UI | DM Sans | 14px | 500 | 1.4 | `text-sm font-medium` | Status badges, price, lead-time note, CTA secondary text |
| Heading | Fraunces | 28px | 400 | 1.2 | `text-[28px] font-normal` | Piece title on detail page |
| Display | Fraunces | 48px | 400 | 1.1 | `text-5xl font-normal` | Gallery page title ("Works" / "Arbejder") |

**Letter-spacing:** Display and Heading use `tracking-tight` (-0.01em). Body and Label use default tracking.
**Weights:** 400 (regular) and 500 (medium) only. No bold in this phase.
**Serif usage:** Fraunces for piece titles and gallery page heading only. All badges, buttons, filter controls, and captions use DM Sans.

Source: Phase 1 UI-SPEC `## Typography`.

## Color

Full palette inherited from Phase 1. Accent usage is extended for this phase.

| Role | Token | Hex | Usage in this phase |
|------|-------|-----|---------------------|
| Dominant (60%) | `--color-linen` | `#F5F0E8` | Page background, detail page background |
| Secondary (30%) | `--color-oat` | `#EDE7D9` | Card background, filter button resting state, detail page right-column background tint |
| Neutral mid | `--color-clay` | `#C4A882` | Card border (hairline), divider between detail columns, filter toggle border |
| Neutral dark | `--color-stone` | `#7A6A58` | Sold badge background, secondary body text, captions, price label, lead-time note |
| Foreground | `--color-ink` | `#2C2418` | Primary body text, piece titles, headings |
| Accent (10%) | `--color-terracotta` | `#A85C3A` | For-sale badge background, "Contact to buy" primary CTA background, active filter tab indicator, focus rings |
| Destructive | `--color-fault` | `#8B3A2A` | Not used in this phase |
| Surface inverse | `--color-ink-surface` | `#1E1A12` | Footer (inherited, no change) |

**60/30/10 split:** linen dominates all surfaces; oat for cards and filter controls; terracotta reserved to the exact elements listed — nothing else.

**Accent (terracotta) reserved for in this phase:**
1. For-sale status badge background (`#A85C3A` with white or linen text)
2. "Contact to buy" primary CTA button background
3. Active filter tab indicator (bottom border or filled tab, not full background)
4. Focus ring outline on all interactive elements (inherited rule from Phase 1)

**Sold badge:** Uses stone (`#7A6A58`) background with linen (`#F5F0E8`) text — not terracotta. Stone signals neutral archival status, not availability.

Source: Phase 1 UI-SPEC `## Color`. Extended accent list for Phase 3 specifics.

## Interaction Contracts

### Grid Layout (GALL-01)

- **Columns:** 3 at ≥1024px, 2 at 640–1023px, 1 below 640px. CSS Grid via `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`.
- **Grid gap:** `gap-6` (24px) on mobile/tablet, `gap-8` (32px) on desktop.
- **Card:** image-only at rest. No text rendered below image. Image fills the card entirely.
- **Image aspect ratio:** 4:5 portrait (`aspect-[4/5]`). Enforced on all grid cards via `aspect-[4/5] overflow-hidden`. This matches the ceramics photography guidance (4:5 portrait hero ratio in CLAUDE.md) and creates consistent Japandi rhythm.
- **Card background:** oat (`--color-oat`). Visible during image load before blur placeholder resolves.
- **Card border:** 1px hairline clay (`--color-clay`), `rounded-none` (no border radius — Japandi restraint).

### Grid Card Hover State

- **Effect:** `scale-[1.015]` on the card image (not the card element), `transition-transform duration-300 ease-out`.
- **No opacity dimming, no overlay, no text reveal on hover.** Japandi restraint — the image speaks for itself.
- **Cursor:** `cursor-pointer` on the entire card.

### Status Badge (D-03)

- **Position:** top-left corner of image (`absolute top-2 left-2`).
- **Style:** pill shape (`rounded-full`), `px-2 py-1`, `text-sm font-medium`.
- **Sold badge:** stone background (`bg-stone`), linen text (`text-linen`).
- **For-sale badge:** terracotta background (`bg-terracotta`), linen text (`text-linen`).
- **No badge** for portfolio-only pieces.
- Badge uses sentence case text.

### For-Sale Filter (D-04, D-05, GALL-03)

- **Controls:** two tab-style toggle buttons: "All" and "For Sale", left-aligned above the grid.
- **URL param:** `?filter=for-sale` for the for-sale view; no param (or param absent) for "All".
- **Active tab:** terracotta bottom border (`border-b-2 border-terracotta`), ink text (`text-ink`), weight 500.
- **Inactive tab:** no border, stone text (`text-stone`), weight 400, underline on hover.
- **Tab padding:** `px-4 py-3` (`16px × 12px`) to meet 44px touch target height.
- **Transition:** `transition-colors duration-150` on text and border color.
- **Implementation note:** Filter toggle is a client component. It reads `searchParams` and calls `useRouter` from `next/navigation` to update the query param. The gallery grid RSC receives `searchParams` as a prop and filters works server-side.

### Detail Page Layout (D-06, D-07, GALL-02)

**Desktop (≥1024px):**
- Side-by-side: left column (primary image, 55% width), right column (title + description + CTA, 45% width).
- Left/right split via `grid grid-cols-[55fr_45fr] gap-8`.
- Right column is vertically centered relative to the primary image height (`content-start` with `pt-8`).
- Additional images (if more than one): full-width below the side-by-side section, displayed in a 2-column grid `grid-cols-2 gap-4`, capped at 800px max-width, centered.

**Mobile (< 1024px):**
- Stacked: primary image full-width, then title/description/CTA below.
- Gap between image and text block: `gap-8` (32px).

**Detail page max-width:** 1024px for the content area (`max-w-screen-lg mx-auto`), inside the standard page horizontal padding. This keeps the detail page from feeling too wide and centers attention on imagery.

**Primary image:** `next/image` with `priority`, `placeholder="blur"`, `sizes="(max-width: 1024px) 100vw, 55vw"`. Natural aspect ratio preserved — no forced crop on detail page.

**Additional images:** `next/image` with lazy load, `placeholder="blur"`, `sizes="(max-width: 640px) 100vw, 50vw"`. `aspect-[4/5]` enforced for consistency.

### Sold Piece Treatment (D-08, D-09, GALL-04)

**Grid card:** stone "Sold" badge at top-left. Image is not desaturated or overlaid.

**Detail page:** Below the description, separated by `mt-8` (32px):
1. "This piece has been sold." — body text (DM Sans 16px, stone color).
2. Ghost CTA button below the sold sentence: "Commission something similar →".
   - Style: `border border-terracotta text-terracotta bg-transparent`, `px-6 py-3`, `text-sm font-medium`.
   - Hover: `bg-terracotta text-linen`, `transition-colors duration-150`.
   - Links to `/[locale]/custom-orders`.

### Focus Management

Inherited from Phase 1 — no changes. All interactive elements: `outline: 2px solid var(--color-terracotta); outline-offset: 2px` on `:focus-visible` only.

### URL Paths

- Gallery route: `/[locale]/gallery` (semantically precise; "gallery" is the browsable collection of works, "works" is an internally-correct but less discoverable slug for first-time visitors)
- Detail route: `/[locale]/gallery/[slug]`
- `generateStaticParams` on the detail page pre-renders all published works at build time.

Source: Phase 3 CONTEXT.md decisions D-01 through D-10, GALL-01 through GALL-04, DSGN-04.
